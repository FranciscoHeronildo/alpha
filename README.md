## Ambiente de Desenvolvimento

Clone o repositório:

```bash
$ cd ~/your/directory/
$ git clone https://github.com/FranciscoHeronildo/alpha.git
$ cd alpha
```

Instale as Dependências:

```bash
$ yarn install
```

Inicie a Aplicação:

```bash
$ yarn dev
```

Acesse a aplicação no navegador pela porta _5173_:

```
http://localhost:5173/
```
